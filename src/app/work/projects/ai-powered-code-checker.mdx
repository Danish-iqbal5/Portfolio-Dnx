# AI Powered Code Checker

An intelligent web app that reviews code using AI models and suggests improvements. Built using Node.js for backend APIs and React for the frontend interface.

## Features

- Automated code quality assessment
- Real-time syntax and style checking
- AI-powered code improvement suggestions
- Support for multiple programming languages
- Clean, modern React-based user interface
- RESTful API architecture with Express.js
- Secure authentication and user management

## Technical Stack

- **Backend**: Node.js, Express.js
- **Frontend**: React, TypeScript
- **AI Integration**: OpenAI GPT API
- **Authentication**: JWT
- **Database**: MongoDB
- **API Documentation**: Swagger/OpenAPI

## Development Process

The project was built with a focus on modularity and scalability. The backend services are containerized for easy deployment, and the frontend implements modern React patterns including hooks and context for state management.

## Impact & Results

This tool helps developers maintain code quality and consistency while learning best practices through AI-powered suggestions. It serves as both a code review tool and a learning platform for improving coding skills.
